# Dotfiles Repository

This repository contains configuration files (dotfiles) for various tools and shells.

## Contents
- `.bashrc` - Bash shell configuration
- `.vimrc` - Vim editor configuration
- `.gitconfig` - Git configuration
- `.tmux.conf` - Tmux configuration
- `install.sh` - Script to install dotfiles by creating symlinks in your home directory

## Installation
1. Clone this repository:
   ```sh
   git clone <repository-url> $HOME/dotfiles
   ```
2. Run the install script:
   ```sh
   cd $HOME/dotfiles
   ./install.sh
   ```
3. Enjoy your configured environment!
